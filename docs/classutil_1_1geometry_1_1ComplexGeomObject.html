◆ d_objFlag

std::vector<std::string> util::geometry::ComplexGeomObject::d_objFlag

Object flag.

Ordering of objects is important. To describe a rectangle with circular hole, we will have d_obj = {rectangle, circle} and have flag = {plus, minux}. This means final object is rectangle - circle
